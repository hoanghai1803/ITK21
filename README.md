## Syllabus

- **Lesson 1 (10.11)**

  - Concepts:

    - Competitive Programming Overview.
    - Complexity.
    - Setup Environment.
    - Debugging.
    - Brute-force.
    - Recursion & Backtracking
    - Branch-N-Bound.
    - Greedy.

  - Resources:
    - [What is Competitive Programming? (William Lin)](https://youtu.be/ueNT-w7Oluw)
